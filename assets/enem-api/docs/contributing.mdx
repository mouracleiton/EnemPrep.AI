Ficamos felizes que você tenha interesse em contribuir com o projeto! Existem várias formas de contribuir, desde reportar um bug, sugerir uma nova funcionalidade, melhorar a documentação, até mesmo enviar um pull request com uma nova funcionalidade.

A principal (e mais fácil) forma de contribuir é deixando sua estrela no [repositório do projeto](https://github.com/yunger7/enem-api). Isso ajuda a aumentar a visibilidade do projeto e a atrair mais contribuidores.

## Reportar um problema

Se você encontrou um problema no projeto, considere [criar uma nova issue](https://github.com/yunger7/enem-api/issues) no repositório do projeto. Forneça o máximo de contexto possível, bem como os passos necessários para reproduzir o problema.

## Contribuir com código

Se você deseja contribuir com código, sinta-se à vontade para enviar um [pull request](https://github.com/yunger7/enem-api/pulls) com novas funcionalidades, melhorias ou correções de bugs.
